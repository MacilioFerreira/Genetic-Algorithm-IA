
-- Como executar o programa
    1 - Abra o terminal na pasta do trabalho e execute o comando:
        python trabalho4.py

    2 - Para alterar os parâmetros do algoritmo, faça:
        1 - Abra o arquivo trabalho4.py
        2 - Na função main(), na parte AG.A_Genetico() passe os parâmetros de acordo com o que deseja.
            2.1 - Ordem dos parâmetros:
                2.1.1 - Elitismo.
                2.1.2 - Mutação.
                2.1.3 - Tamanho da população.
                2.1.4 - Quantidade de iterações.
                2.1.5 - Quantidade de enfermeiros(a).
                2.1.6 - Quantidade de turnos.


-- Sistema utilizado e requisitos para rodar o programa
    - Linux - 32 bits
    - IDE utilizada pyCharm (Código pode ser aberto em um editor de texto qualquer).
    - Ter o python instalado.
